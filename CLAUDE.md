## What is dota-meta?

A Dota 2 bracket-specific meta intelligence tool. It fetches hero stats from the OpenDota API, analyzes win rates per skill bracket, and produces:

1. **Go CLI** that generates Reddit-formatted markdown posts showing best heroes, sleeper picks, trap picks, and bracket deltas.
2. **Static website** (GitHub Pages) with interactive sortable tables and bracket selector.

This is a **free and open source** community tool. It is not a paid product. It builds the Narrowcast brand in r/DotA2 (1.3M+ members) and drives traffic to the static site.

## Tech stack

- **Go** (stdlib only, no external deps) for the CLI
- **Vanilla HTML/JS/CSS** for the static site (no React, no build tools)
- **GitHub Pages** for hosting

## Dev environment

This project uses **Nix flakes**. The `flake.nix` provides Go and Node.js. Run `direnv allow` to activate the dev shell automatically.

## Project structure

```
dota-meta/
  cmd/dota-meta/main.go        -- CLI entry point
  internal/
    api/api.go                  -- Fetch + parse OpenDota heroStats
    analysis/analysis.go        -- Core analysis logic
    format/
      reddit.go                 -- Reddit markdown formatter
      json.go                   -- JSON output formatter
  site/
    index.html                  -- Static site
    style.css                   -- Styles
    app.js                      -- Vanilla JS
    data.json                   -- Generated by CLI
  testdata/herostats.json       -- Test fixture
```

## Data source

Single API call to `https://api.opendota.com/api/heroStats`. No API key required. Zero running cost.
